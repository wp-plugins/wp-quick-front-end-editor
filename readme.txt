 Edit Posts,Custom posts and Pages content including Featured image from frontend with visual editor.

== Description ==

WP Quick FrontEnd Editor plugin will allow you to change and save contents of post and pages by using WYSIWYG editor from frontend of your site without going to Dashboard for little change.You can format your text as you want.
You can also add images from wordpress gallery by drag and drop and by using upload button.You can add,remove and change featured image.

<h2>Features</h2>
<ul>
	<li> Edit content with visula responsive editor </li>

	<li> Responsive  </li>

== Installation ==
1. Go to plugins in your dashboard and select \'add new\'
2. Search for \'WP Front End Editor\', install and activate it
3. Now view your posts and pages and enable plugin by clicking on enable button.Now edit your content and images
4. Click save button when over with changes and your post or pages will be updated according to changes
